### 📌 MVP

| 기능 | 내용 |
| --- | --- |
| 회원가입/로그인 | 사용자/관리자 구분 및 계정 생성, 삭제 수정 기능 |
| 위치 조회 | 사용자 위치 기반 주변 가게 및 이벤트 목록 조회 |
| 가게 | 가게 이름, 가게 위치 확인할 수 있는 기능 |
| 이벤트 | 이벤트 내용, 이벤트 기간, 이벤트 혜택 등 다양한 이벤트를 확인할 수 있는 기능 |
